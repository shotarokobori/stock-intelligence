# 📊 日本株インテリジェンス 自動配信システム

毎日のニュースとYouTube動画をAIが分析し、**日本株運用で勝つための統合レポート**を毎朝Gmailに届けます。

---

## 届くレポートの内容

| セクション         | 内容                               |
| ------------------ | ---------------------------------- |
| 今日の3行サマリー  | 全体像を3文で把握                  |
| 各ソースの要点     | 記事・動画の解説＋専門用語注釈     |
| 統合考察（詳細）   | 複数ソースを横断した深い分析と論拠 |
| 今後の注目ポイント | 1〜2週間のカレンダー＋影響予測     |
| 日本株見通し       | 日経平均↑↓・注目業種・おすすめ銘柄 |

## セットアップ手順（初回のみ）

### ステップ 1：Pythonのインストール確認

コマンドプロンプト（Windowsキー → 「cmd」と入力 → Enter）で以下を実行：

```
python --version
```

`Python 3.10.x` のように表示されればOKです。表示されない場合は https://www.python.org/ からインストールしてください。

---

### ステップ 2：必要なライブラリのインストール

コマンドプロンプトで、このフォルダに移動してから実行：

```
cd C:\Users\あなたのユーザー名\stock_intelligence
pip install -r requirements.txt
```

---

### ステップ 3：`.env` を作成してAPIキーを登録

1. `stock_intelligence` フォルダの `.env.example` を開きます。
2. 内容をそのままコピーして新しいファイル `.env` を作成します。
3. `ANTHROPIC_API_KEY` に `sk-ant-...` 形式のAnthropic APIキーを貼り付けます。
4. `GMAIL_SENDER_EMAIL` / `GMAIL_SENDER_PASSWORD` / `GMAIL_RECIPIENT_EMAILS` も必要に応じて編集します。

> `.env` は `.gitignore` ですでに除外されています。GitHubへ公開しないようにしてください。

---

### ステップ 4：Anthropic APIキーとGmail設定の確認

1. `ANTHROPIC_API_KEY` を `.env` に書き込みます。
2. `GMAIL_SENDER_EMAIL` に送信元メールアドレスを設定します。
3. `GMAIL_SENDER_PASSWORD` にGoogleアプリパスワードを設定します。
4. `GMAIL_RECIPIENT_EMAILS` にカンマ区切りで受信先メールアドレスを設定します。

> 通常のGmailパスワードは使えません。「アプリパスワード」が必要です。

1. Googleアカウント（myaccount.google.com）→「セキュリティ」
2. 「2段階認証プロセス」を有効にする（まだの場合）
3. 「アプリパスワード」→「メール」「Windowsコンピュータ」を選択 → 「生成」
4. 表示された16文字をコピー
5. `config.json` の `"sender_password"` に貼り付ける

---

### ステップ 5：YouTube Data APIキーの取得（任意・無料）

YouTubeチャンネルの動画も分析したい場合のみ設定します。

1. https://console.cloud.google.com にアクセス
2. 新しいプロジェクトを作成
3. 「APIとサービス」→「ライブラリ」→「YouTube Data API v3」を有効化
4. 「認証情報」→「APIキーを作成」
5. `config.json` の `"youtube"` → `"api_key"` に貼り付ける

> 無料枠：1日あたり10,000クォータ（このシステムでは1日100程度しか使わないので問題なし）

---

### ステップ 6：動作テスト

```
python main.py --test
```

エラーなく完了すると `data/` フォルダに `test_report_日付.html` が保存されます。
ブラウザで開いて内容を確認してください。

---

### ステップ 7：Windowsタスクスケジューラで毎朝自動実行

1. スタートメニュー →「タスクスケジューラ」を検索して起動
2. 右側「基本タスクの作成」をクリック
3. 名前：`日本株インテリジェンス`
4. トリガー：「毎日」→ 開始時刻 `07:00:00`
5. 操作：「プログラムの開始」
   - プログラム：`python`
   - 引数：`C:\Users\あなたのユーザー名\stock_intelligence\main.py`
   - 開始（作業ディレクトリ）：`C:\Users\あなたのユーザー名\stock_intelligence`
6. 「完了」をクリック

## ソースの追加・削除方法

`sources.json` をテキストエディタ（メモ帳でOK）で開いて編集します。

### RSSフィードを追加する場合

```json
{
  "name": "追加したいサイトの名前",
  "url": "RSSフィードのURL",
  "enabled": true,
  "memo": "メモ"
}
```

### YouTubeチャンネルを追加する場合

チャンネルIDの調べ方：YouTubeのチャンネルページURLに含まれる `UC...` で始まる文字列

```json
{
  "name": "チャンネル名",
  "channel_id": "UCxxxxxxxxxxxxxxx",
  "enabled": true,
  "memo": "メモ"
}
```

### 一時的に無効にする場合

`"enabled": true` を `"enabled": false` に変更するだけです。

## よくある問題と対処法

| 症状                  | 対処法                                                       |
| --------------------- | ------------------------------------------------------------ |
| `ModuleNotFoundError` | `pip install -r requirements.txt` を再実行                   |
| Gmail送信エラー       | アプリパスワードが正しいか確認。2段階認証が有効か確認        |
| 記事が0件             | RSSのURLが正しいか確認。インターネット接続を確認             |
| YouTube動画が取れない | APIキーが正しいか確認。APIが有効化されているか確認           |
| Claude APIエラー      | APIキーが正しいか確認。Anthropicアカウントに残高があるか確認 |

---

## ログの確認

`logs/system.log` にすべての実行ログが記録されます。
問題が起きたらこのファイルを確認してください。
